1) Modifying an area

Before starting to build your area, I strongly suggest you read this article by Urgit.

First of all, type "aedit" in your area. You should see something like this:

[Area-Edit]
Name:     [ 112] Example Area
File:     ../area/example.are
Vnums:    [1000-1999]
Min-Max:  [0-0]
Age:      [0]
Players:  [1]
Security: [9]
Builders: [builder]
Credits : [{??} builder Example Area]
Flags: [changed added noaquest]
Customs:
        Time:
            Daybreak [05:00] - [The day has begun.]
            Sunrise [06:00] - [The sun rises in the east.]
            Sunset [19:00] - [The sun slowly disappears in the west.]
            Darkness [20:00] - [The night has begun.]
        Weather:
            Get's Cloudy - [The sky is getting cloudy.]
            Become's Clear - [The clouds disappear.]
            Begin's Raining - [It starts to rain.]
            Stop's Raining - [The rain stopped.]
            Begin's Lighning - [Lightning flashes in the sky.]
            Stops Lightning - [The lightning has stopped.]

To change the name of the area, I would type "name [new area name]", e.g. "name Caverns under Proglu".
To set the suggested level range for the area, type "minmax [min lvl] [max lvl]", e.g. "mixmax 1 91".
To set who can build in this area, type "builder [name]", e.g. "builder Salmissra".
Credits is what appears when you type "area", and to set that type "credit [whatever]".

Now for time and weather:

Weather - "weather (weather type) (string)"
Weather types - cloudy, cloudless, rain, stoprain, lightning, stoplightning

Time - "time (time) (string)"
Times - daybreak, sunrise, sunset, dark

weather cloudy C`&loud``s begin to gather, `8darkening`` the sky.
