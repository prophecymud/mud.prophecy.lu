9)  Modifying the object

Name:        [no name]

You can set the name of the object by doing "name blue checkered cotton shirt".

The name is what you will type when you attempt to pick up, wear etc. something, so please use the same words in "name", "short desc" and "long desc", as this makes life easier for the player. Keep the number of words in "name" between 3-4. Do not include "a", "an" or "the", or use colour.

Type:        [trash]

To set the type of object it is, type "type <type>". Here is the ? file for type:

? type

light              scroll             wand               staff
weapon             treasure           armor              potion
furniture          trash              container          drinkcontainer
key                food               money              boat
npccorpse          fountain           pill               protect
map                portal             warpstone          roomkey
gem                jewelry            jukebox            keyring

They are pretty self-explanatory.

Level:       [    0]

Set the level of the object by typing "level <number>". This sets the level of the object to be worn. Bear in mind that players can wear equipment up to 30 levels above their own level.

Wear flags:  [none]

Set this by typing "wear <wear slot>". This determines which slots the object can be worn on. If the object is supposed to be in the player's inventory, you must include "take".

Extra flags: [none]

To set the different flags on an object, type "extra <flag type>". Setting extras will determine the object's behaviour, for example anti-good on an object makes the object zap any good aligned players who try to wear the object. Personal taste here: align flags aren't useful except to advantage certain aligns in PK, and they aren't really RP (oh no, Belgarion can't disguise himself as a Grolim because it's anti_good!).

Race flags:  [none]
Class flags: [none]

I personally feel that race/class flags should not be widely used, but only used in very, very special pieces of equipment. To set a race/class flag, type "race <race name>" or "class <class name>".

Material:    [unknown]
Condition:   [  100]

These flags don't really have a use. Some builders like setting materials, so for example "material cashmere_and_silk" or "material wood". Do not use spaces in "material" since that'll cause BP to crash. Do not change condition, since we haven't coded anything for the condition of items.

Weight:      [    0]

The weight of the object, multiplied by 1/10 in pounds. So an object with weight 20 would only weigh 2 pounds in the game. Everything has a weight, so at least set it to 1. To set it, type "weight <number>".

Cost:        [    0]

Pretty self-explanatory - the cost/price of the item. The number you enter is in silver, so if you want to make the object worth 500 gold remember to multiply by 100, so that it will be "cost 50000".

Short desc:  (no short description)

This is the description that shows when you wear the item, i.e. what you see when you type "equip" or "inventory", e.g. 'a long sword'. This can be coloured, but do not capitalise the short desc. To set it, type "short <short desc>"

Long desc:
     (no description)

Long descs show when you look at the object (unless it has an extra desc) or when it is on the ground, e.g. "A long sword lies here." These should be capitalised, and a fullstop should be added.
