First of all I need to make the mobile a shopkeeper. I type "shop assign" in medit, then I type "shop profit [buying] [selling]". If I set the shop profit to "shop profit 90 110", the shopkeeper will buy things at 90% of the value and sell at 110% of the value.
        e.g. An apple costs 10 silver. So the shopkeeper buys at 9 silver and sells at 11 silver.
"Shop type [item type]" will determine what kind of shop it is, e.g. "shop type furniture" makes the shop a furniture shop. The shopkeeper will only buy and sell furniture type items. "Shop hours [opening] [closing]" will set the times the shop is open, so setting it to "shop hours 0 23" means the shop will be open 24-hours a day. "Shop remove" will make a shopkeeper an ordinary mobile again.
