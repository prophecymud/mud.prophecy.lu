13)  Creating a mobile

To create a mobile, type “medit create <vnum of mobile>”. In this example, the mobile vnum will be 1000.

medit create 1000

[Mob-Edit]
Mobile created.

To view more information, press the enter key, or type "show". You will see the following:

show
