So the shop is south of the tunnel. To create a two way link, I would type "redit 1001" to edit room 1001 (tunnel), and then type "south link 1000". To create a one way link, type "<direction> room <vnum of destination room>", e.g. "south room 1000". A exit must not exist beforehand.

When I press enter I will see this:

redit 1001
south link 1000

[Exits: north]

We can set different exit flags, for example creating locked, unpickable doors. To do that, I would have type the following:

south door
south locked
south nolock
south nopass
south key <vnum of key object>

Please note that any object can be set to be the key, but for the players' sake (so that they can fit it on their keyrings) remember to set the object type to "key".
