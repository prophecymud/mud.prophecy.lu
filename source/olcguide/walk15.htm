15) Mob descriptions

It is suggested that you add a description to each mobile you create, as it would certainly make an area more interesting than looking at a mobile and just seeing: "You see nothing special about him/her". Also, it may be useful in miniquests, as you can hide subtle hints in a mob description (e.g. needing you to deliver a letter to a specific person). Here are a list of following things you can describe:

  • race and gender
  • hair colour, hair length
  • skin colour
  • general height (don't say "six foot four", since players don't carry around rulers to measure people's heights)
  • stature: fat? thin? muscular?
  • clothing (general, e.g. whether it's clean, tattered etc)
  • expression: happy? sour? angry?
  • posture: slouching, standing at attention etc.
  • scars/birthmarks, if any prominent ones
