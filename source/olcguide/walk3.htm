3)  Modifying the room

Name:       []

To change the name of the room, type “name `8Inside a `4s`$ho`4p``”.

Sector:     [inside]

As you can see, sector is set to "inside", which is correct, as shops are indoors. If you want to set the sector to something else, for example "mountain" or "field", type the sector flag by itself, i.e. "mountain."

Room flags: [none]

Now for the room flags. When I type "? room", I see this:

? room

dark               no_mob             indoors            no_teleport
no_clanrecall      no_quit            private            safe
solitary           pet_shop           no_recall          imp_only
gods_only          heroes_only        newbies_only       law
nowhere            wizard_only        stable             no_magic
nogate             nologon            nostatus           saveover

The room will need to be "dark", since Ulgos can only stand the darkness, and we also need "indoors", which means you cannot cast certain weather spells in the room, or see the weather. If you want to remove an existing flag, for example "no_mob", type "no_mob".

Health recovery:[100]
Mana recovery  :[100]

Modifying the health and mana regeneration rates will not be necessary for a shop. Setting the rates to 0 means players will not heal every tick, and 100 is the normal healing rate. If for example it was a healer's room, you may change to to 150 (i.e. 50% faster healing rate) by typing "heal 150" and "mana 150".

Owner :     []
Characters: [builder]
Objects:    [none]

Do not set the owner flag to anything, as that means only the owner of the room and immortals can enter the room. However, you may set owner flags in clan rooms. The characters and objects flags change according to who/what is in the room, and do not need to be set.
