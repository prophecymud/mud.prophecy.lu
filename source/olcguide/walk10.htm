10)  Equipment

To make equipment, I will set the "type" to armor, by typing "type armor". In this example we will be making a blue, checkered shirt. This is what I will see:

type armor

So I will type the following:

name blue checkered shirt
level 31
wear take
wear torso
extra burnproof
material cotton
weight 1
cost 50
short a `4b`$lu`4e`7, checkered s`&hir``t
long A blue, checkered shirt lies here.

[v3] Ac exotic       [0]

Now for the v0 - v3. Before setting the ac (i.e. "armour class"), remember to read "help eq" and read about the point system. To set the different armour classes, type "v0 <number>", e.g. "v0 5". The number should be positive.

And finally statistics: hp, mana, movement, strength, intelligence etc. How do we set them? We use "addaffect". By typing "? apply", we can see what kind of effects we can add to a piece of equipment:

For example, if I wanted to add 1 wisdom to the shirt, I would type "addaffect wisdom 1". To add a spell I would type "addaffect haste 1", i.e. the haste spell is at level 1. So now we have this:
