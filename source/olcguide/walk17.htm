17) Creating a prog

There are currently two kinds of programs: mprogs and oprogs, but they are both created from 'pedit'. Mprogs stand for mobile programs, which means programs a mobile will execute if triggered. Oprogs is for objects, and some of the commands are buggy, so exercise caution when using oprogs.

To create a program, type “pedit create <vnum of program>”. In this example, the room vnum will be 1000.

pedit create 1000

[Mobprog-Edit]
MobProgram created.

To view more information, press the enter key, or type 'show' (enter key is faster). You will see the following:

show

[Mobprog-Edit]

Vnum:       [1000]

Program Code:
-------------


-------------

To actually code the program, I will need to type "code". This will enter me into the append mode. I will then type the program, and then leave append mode. I cannot explain how to write a program - you need to read the mprog and oprog sections of the OLC guide and more importantly, look at the examples.

code

[Mobprog-Edit]
-=======- Entering APPEND Mode -========-
Type .h on a new line for help
Terminate with a ~ or @ on a blank line.
-=======================================-

>

if race $n ulgo
or carries $n 1002
say Yad ho, groja UL.
mob echoat $n $I says '`%Welcome to my store. Please do not touch any of the drawers, as some herbs are poisonous.``'
mob echoaround $n $I says '`%Gral matia dosart. Rolga krash ni saalko, par na ros lupakjo.``'
else
say Yad ho, groja UL.
say Gral matia dosart. Rolga krash ni saalko, par na ros lupakjo.
endif
.s

Vnum:       [1000]

Program Code:
-------------

if race $n ulgo
or carries $n 1002
say Yad ho, groja UL.
mob echoat $n $I says 'Welcome to my store. Please do not touch any of the drawers, as some herbs are poisonous.'
mob echoaround $n $I says 'Gral matia dosart. Rolga krash ni saalko, par na ros lupakjo.'
else
say Yad ho, groja UL.
say Gral matia dosart. Rolga krash ni saalko, par na ros lupakjo.
endif

-------------

This means the mobile will echo to the target "[mobile name] says 'Welcome to my store...'" and echo to everyone else "[mobile] says 'Gral matia dosart...'" if the target is an Ulgo, or carries the object with vnum 1002. If it is neither an Ulgo or the object bearer, the mobile will say 'Gral matia dosart...'.

Now I need to add it to the shopkeeper (vnum 1000). To do this, I will enter medit, and type "addmprog 1000 grall 100". This means I will be adding a mprog of vnum 1000 to trigger 100% of the time someone enters the mobile's room, regardless of whether the mobile is blinded, fighting etc. If I had set it to greet, then the mobile will trigger 100% of the time if it is not busy, and if the target is visible to the mobile. To give you examples, grall will be Harakan and greet will be Unicorn (Ulgoland).

medit 1000
